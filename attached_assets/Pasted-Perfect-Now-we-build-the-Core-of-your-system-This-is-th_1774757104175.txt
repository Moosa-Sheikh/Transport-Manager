🔷 STEP 1: UPDATE DATABASE SCHEMA

Open database/schema.sql
Add the following table:

1️⃣ trips
CREATE TABLE trips (
    id SERIAL PRIMARY KEY,
    trip_date DATE NOT NULL,
    truck_id INTEGER NOT NULL REFERENCES trucks(id) ON DELETE RESTRICT,
    driver_id INTEGER NOT NULL REFERENCES drivers(id) ON DELETE RESTRICT,
    from_city_id INTEGER NOT NULL REFERENCES cities(id) ON DELETE RESTRICT,
    to_city_id INTEGER NOT NULL REFERENCES cities(id) ON DELETE RESTRICT,
    status VARCHAR(20) CHECK (status IN ('Open','Closed')) DEFAULT 'Open',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Important:

Use foreign keys
Do not allow cascade delete
Restrict delete if trip exists

Run migration safely.
