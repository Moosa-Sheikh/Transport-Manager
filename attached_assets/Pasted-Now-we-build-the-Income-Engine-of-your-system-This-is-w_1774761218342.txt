🔷 STEP 1: UPDATE DATABASE SCHEMA

Open database/schema.sql
Add:

1️⃣ trip_loads
CREATE TABLE trip_loads (
    id SERIAL PRIMARY KEY,
    trip_id INTEGER NOT NULL REFERENCES trips(id) ON DELETE CASCADE,
    bilty_number VARCHAR(100) NOT NULL,
    customer_id INTEGER NOT NULL REFERENCES customers(id) ON DELETE RESTRICT,
    item_description VARCHAR(150),
    weight NUMERIC(12,2),
    freight NUMERIC(12,2) DEFAULT 0,
    loading_charges NUMERIC(12,2) DEFAULT 0,
    unloading_charges NUMERIC(12,2) DEFAULT 0,
    broker_commission NUMERIC(12,2) DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Important:

One trip → many loads
If trip deleted → loads auto delete (CASCADE allowed here)

🔷 STEP 6: UPDATE TRIP LIST PAGE

Add new column:

Trip Income

Use LEFT JOIN with aggregated subquery:

Example structure:

LEFT JOIN (
    SELECT trip_id,
           SUM(freight + loading_charges + unloading_charges - broker_commission) AS income
    FROM trip_loads
    GROUP BY trip_id
) income_data ON t.id = income_data.trip_id

If no loads → income = 0
